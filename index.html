HTML & CSS Reference Page

This website is to help our future projects.

It contains useful information on HTML, CSS and Termianl such as commands and shortcuts.

It also includes useful websties for future reference.

HTML

Stands for HyperText Markup Language. HTML is the most basic building block of the Web. It describes and defines the content of a webpage.

CSS

Stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

Terminal

The word "terminal" comes from early computer systems that were used to send commands to other computers.The purpose of a terminal is not to process information (like a typical computer), but to send commands to another system.
